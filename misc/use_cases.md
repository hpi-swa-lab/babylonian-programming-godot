# Use cases

## Live debugging

Das Halo tool kann vor allem dafür verwendet werden, ein Spiel während der Laufzeit zu debuggen. Es lassen sich Werte von Eigenschaften der Objekte zur Laufzeit auslesen und manupulieren.

### Szenarien

1. Der Entwickler baut einen Platformer. Er möchte eine Platform in genau der richtigen Entfernung platzieren. Anstatt das Spiel für jede Änderung zu beenden und in den Editor zu wechseln, kann er die Platform in-game mit dem Halo bewegen. Hat er die richtige Position gefunden, kann er den Inspector öffnen, die Position kopieren und im Editor einfügen.
